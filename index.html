Apa itu coding ?

coding adalah proses menulis instruksi atau perintah dalam bahasa pemrograman untuk komputer agar dapat memahami dan menjalankan tugas tertentu, seperti membuat aplikasi, situs web, atau game. Dengan kata lain, koding adalah cara manusia "berbicara" dengan komputer untuk memberikan instruksi, dan hasil dari proses ini adalah perangkat lunak yang kita gunakan sehari-hari.

Berikut beberapa bahasa pemograman yang sering dipakai :

Java script

Php

Pyton

C++
